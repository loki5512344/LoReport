# Loreport

[![License: GPL v3](https://img.shields.io/badge/License-GPLv3-blue.svg)](https://www.gnu.org/licenses/gpl-3.0)
[![Minecraft Version](https://img.shields.io/badge/Minecraft-1.19.2--1.21.11-green.svg)](https://papermc.io/)
[![Java Version](https://img.shields.io/badge/Java-21+-orange.svg)](https://adoptium.net/)

Современная и производительная система репортов для Minecraft серверов с поддержкой Discord интеграции и интуитивным графическим интерфейсом.

## ✨ Возможности

- 📝 **Система репортов** - удобная отправка жалоб на нарушителей
- 🎨 **Графический интерфейс** - интуитивный GUI для управления репортами
- 📊 **Статистика и история** - детальная информация по каждому игроку
- 🔔 **Discord интеграция** - мгновенные уведомления через webhook
- 🗄️ **Поддержка БД** - SQLite для малых серверов, PostgreSQL для крупных
- 🌐 **Локализация** - полностью настраиваемые сообщения
- ⚡ **Оптимизация** - асинхронная работа с БД, минимальное влияние на TPS
- 🔒 **Безопасность** - защита от SQL-инъекций, валидация данных

## 🚀 Быстрый старт

1. Скачайте последнюю версию из [Releases](https://codeberg.org/loki5512344/Loreport/releases)
2. Поместите `Loreport-x.x.x.jar` в папку `plugins/`
3. Перезапустите сервер
4. Настройте `plugins/Loreport/config.yml`

## 🎯 Совместимость

- **Minecraft**: 1.19.2 - 1.21.11
- **Серверное ПО**: Paper, Purpur, Pufferfish и другие Paper-форки
- **Java**: 21+
- **База данных**: SQLite 3.47+ или PostgreSQL 12+
