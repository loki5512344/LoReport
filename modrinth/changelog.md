# Changelog

## v1.1.0 - Улучшенная версия

### ✨ Новые возможности
- Поддержка Minecraft 1.19.2 - 1.21.4
- Улучшенная конфигурация с новыми опциями
- Добавлены алиасы команд: `/rep`, `/reports`
- Система уведомлений администраторов в игре
- Настраиваемые ограничения на количество репортов
- Улучшенный GUI с настройками отображения

### 🔧 Улучшения
- Оптимизирована производительность базы данных
- Улучшена совместимость с различными версиями Minecraft
- Обновлены зависимости для лучшей стабильности
- Добавлена поддержка Java 17+ (вместо Java 21)
- Улучшена обработка ошибок

### 🐛 Исправления
- Исправлены проблемы с кодировкой сообщений
- Улучшена стабильность Discord webhook
- Исправлены мелкие баги в GUI

### 📚 Документация
- Полная документация в папке `docs/`
- FAQ с ответами на частые вопросы
- Руководство по API для разработчиков
- Подробное описание конфигурации

## v1.0.0 - Первый релиз

### ✨ Основные возможности
- Система репортов с командой `/report`
- GUI интерфейс для администраторов
- Статистика репортов по игрокам
- Интеграция с Discord через webhook
- Поддержка SQLite и PostgreSQL
- Настраиваемые сообщения
- Система прав доступа
