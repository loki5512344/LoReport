# Информация о проекте для Modrinth

## Основная информация
- **Название**: Loreport
- **Краткое описание**: Современная система репортов для Minecraft серверов с Discord интеграцией
- **Категории**: Management, Utility, Server Management
- **Лицензия**: GPL-3.0
- **Тип проекта**: Plugin (Server-side)

## Поддерживаемые версии

## Зависимости
Плагин не требует дополнительных зависимостей, все необходимые библиотеки включены в jar файл.

## Рекомендуемые плагины
- **LuckPerms** - для расширенного управления правами
- **PlaceholderAPI** - для интеграции с другими плагинами
- **DiscordSRV** - для дополнительной интеграции с Discord

## Галерея изображений
1. GUI интерфейс с репортами
2. Пример Discord уведомления
3. Статистика игрока
4. Конфигурационный файл

## Инструкции для загрузки
1. Выберите версию плагина, совместимую с вашей версией Minecraft
2. Скачайте файл `Loreport-x.x.x.jar`
3. Поместите файл в папку `plugins/` вашего сервера
4. Перезапустите сервер
5. Настройте конфигурацию в `plugins/Loreport/config.yml`
6. При необходимости настройте Discord webhook
